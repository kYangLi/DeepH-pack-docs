# Getting the Code

DeepH-pack is a proprietary software that requires proper licensing for use. Follow these steps to obtain access:

## Software Acquisition

### 1. Visit the Official Website

Go to the [DeepH-pack application portal](https://ticket.deeph-pack.com/?language=en) to start the acquisition process.

### 2. Complete the Application Form

- Fill in your personal and institutional details
- Provide your academic or professional affiliation
- Agree to the software license terms

### 3. Wait for Approval

- Applications are typically processed within 1-2 business days
- You will receive a confirmation email with further instructions

### 4. Access Your Software

Upon approval:

1. You will receive a unique download link
2. Access credentials will be provided
3. Download the software package

## Note for Different User Groups

- **Academic Researchers**: Educational licenses are available
- **Open Source Contributors**: Check our [GitHub repository](https://github.com/kYangLi/DeepH-pack-docs) for development versions

## Troubleshooting Acquisition Issues

If you encounter problems during the application process:

- Check your spam folder for approval emails
- Ensure all required fields are completed
- Contact <deeph-pack@outlook.com> for assistance
